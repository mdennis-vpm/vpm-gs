"Slide" animation, enabled by adding the class .vpa-slide-edge-(direction) to an element. There are three options:

  • .vpa-slide-edge-top
  • .vpa-slide-edge-right
  • .vpa-slide-edge-left

Element will start just beyond the viewport edge of the given direction.
When triggered, the element will slide into place from that starting point.
"edge-bottom" is not an option, due to causing scrollbar/scroll position issues.
All slide classes can be combined.

(In tandem with .vpa-out class, element will instead slide OUT to the given edge of the viewport.)
