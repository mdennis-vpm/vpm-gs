"Coinspin" animation, enabled by adding the class .vpa-coinspin to an element. There are two options:

  • .vpa-coinspin-x
  • .vpa-coinspin-y

When triggered, the element will spin once along the center of the given axis.
Both coinspin classes can be combined.

(The .vpa-out class has no impact on this property)
