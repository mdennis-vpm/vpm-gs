"Delay" modifier, enabled by adding the class .vpa-delay-(delay) to an element. There are 4 total options:

  • .vpa-delay-none = no delay
  • .vpa-delay-short = 0.1s delay
  • .vpa-delay-long = 0.5s delay
  • .vpa-delay-xlong = 1s delay

Without using any of these, the default delay of 0.2s is used.

Changes the delay between the triggering and playing of the animation, from default, to the given delay.

(The .vpa-out class has no impact on this property)
