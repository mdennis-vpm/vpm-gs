"Slide" animation, enabled by adding the class .vpa-slide-container-(direction) to an element. There are four options:

  • .vpa-slide-container-top
  • .vpa-slide-container-right
  • .vpa-slide-container-bottom
  • .vpa-slide-container-left

Element will start out flush against the edge of the given direction of its container.
When triggered, the element will slide into place from that starting point.
All slide classes can be combined.

(In tandem with .vpa-out class, element will instead slide OUT to the given edge of its container.)
