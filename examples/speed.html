"Speed" modifier, enabled by adding the class .vpa-(speed) to an element. There are 6 total options:

  • .vpa-xfast = 0.1s duration
  • .vpa-fast = 0.25s duration
  • .vpa-slow = 1s duration
  • .vpa-xslow = 2s duration
  • .vpa-xxslow = 3s duration
  • .vpa-xxxslow = 5s duration

Without using any of these, the default duration of 0.5s is used.

Changes the duration of the animation from default, to the given one.

(The .vpa-out class has no impact on this property)
