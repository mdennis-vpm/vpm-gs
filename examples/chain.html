"Chain" animation enhancement, enabled by adding the class .vpa-chain to elements with an animation class.

With .vpa-chain, any elements with animation classes, which are directly adjacent siblings in the html, will animate with progressively larger offsets than the element before it. This gives a chaining animation effect.

Note: This offset is NOT applied on mobile (viewport width smaller than 992px).

(The .vpa-out class has no impact on this property)
