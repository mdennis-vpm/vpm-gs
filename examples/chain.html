"Chain" animation enhancement, enabled by adding the class .gs-chain to elements with an animation class.

With .gs-chain, any elements with animation classes, which are directly adjacent siblings in the html, will animate with progressively larger offsets than the element before it. This gives a chaining animation effect.

Note: This offset is NOT applied on mobile (viewport width smaller than 992px).
