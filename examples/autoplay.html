.vpa-autoplay
.vpa-slide-top

"Autoplay" animation trigger, enabled by adding the class .vpa-autoplay to an element.

Sets the animation to trigger automatically, resulting in immediately playing on page load.

Note: There should never be a need to use this class -- autoplay is applied by default when no other trigger class is provided.

(The .vpa-out class has no impact on this property)
