"Slide" animation, enabled by adding the class .vpa-slide-(direction) to an element. There are four options:

  • .vpa-slide-top
  • .vpa-slide-right
  • .vpa-slide-bottom
  • .vpa-slide-left

Element will start out offset by 75px in the given direction.
When triggered, the element will slide into place from that starting offset.
All slide classes can be combined.

(In tandem with .vpa-out class, element will instead slide OUT in the given direction.)
